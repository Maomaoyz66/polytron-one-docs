## Overview

相机选择用于把摄像头分配到指定网格。

## Workflow

1. 点击要添加相机的网格，右侧显示相机列表。
2. 使用相机 ID 或名称搜索相机。
3. 查看剩余插槽数量。
4. 添加、移除或清空网格中的相机。
5. 使用自动填充按列表顺序填满所有网格。

## 功能说明

- 选择相机：被选中的网格以蓝色边框突出显示。
- 搜索相机：使用相机 ID 或名称查找特定相机。
- 添加相机：将所选相机分配到高亮网格。
- 清除所有相机：移除当前布局中的全部相机。
