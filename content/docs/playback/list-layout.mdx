## Overview

列表布局适合按条件检索并下载录像。

## 功能说明

- 按摄像头 ID 搜索。
- 按日期范围和录像类型筛选。
- 全选当前页面视频。
- 选择单个视频。
- 批量下载所选视频。
- 下载单个视频。
- 分页浏览回放结果。
