## Overview

系统设置用于配置语言、主题、账户信息和存储策略。

## 显示设置

- 语言：英语 / 中文。
- 系统主题：浅色 / 深色。

## 账户设置

- 展示当前账号绑定邮箱。
- 提供编辑密码入口，支持账号密码修改。

## 存储设置

- 最大存储量。
- 视频保留期限，以天为单位。
- 日志保留期限，以天为单位。
- 可配置最大存储容量。
- 支持录像和日志保留天数设置。
- 支持本地存储以及基于 S3 的网络存储启用或关闭。
