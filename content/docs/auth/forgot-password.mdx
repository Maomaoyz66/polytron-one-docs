## Overview

如果忘记密码，用户可以通过电子邮件找回。

## Workflow

1. 点击登录界面的“忘记密码”。
2. 输入与账户关联的电子邮件地址。
3. 查看邮件并输入验证码 Code。
4. 设置新密码并确认。
5. 使用新密码重新登录。

## Notes

邮箱必须与账户绑定。验证码验证通过后才能设置新密码。
